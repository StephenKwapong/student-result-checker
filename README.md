# 🎓 Student Result Checker

A simple Python-based system that calculates student results, averages, grades and pass/fail status based on core school subjects.

---

## 👨‍💻 Author
**Stephen Kwapong**

---

## 📌 Project Description
This project is designed to simulate a real school grading system. It allows a teacher or user to input student scores for different subjects and automatically calculates:

- Total score
- Average score
- Grade (A–F)
- Pass/Fail status

---

## 📚 Subjects Included
- Mathematics  
- English Language  
- Integrated Science  
- Social Studies  
- R.M.E  
- Creative Arts  
- Career Technology  
- Ghanaian Language (Akuapim Twi)

---

## ⚙️ How It Works
1. Enter student name  
2. Enter scores for each subject  
3. The system calculates:
   - Total score
   - Average score
   - Grade
   - Pass/Fail result  

---

## 🧮 Grading System
- 80 – 100 → A  
- 70 – 79 → B  
- 60 – 69 → C  
- 50 – 59 → D  
- Below 50 → F
